# 什么是大语言模型(LLM)

## 定义
大语言模型(LLM, Large Language Model)是一类基于深度学习的人工智能模型，能够理解和生成自然语言文本。

## 发展历史
- 2018年:BERT、GPT等模型提出，开启预训练-微调范式
- 2020年:GPT-3发布，参数量大幅提升，能力显著增强
- 2023年:Llama、ChatGPT等模型推动LLM应用普及

## 代表性模型
- OpenAI GPT系列
- Google BERT、PaLM
- Meta Llama
- 国内:文心一言、通义千问等
